# EduTrack - School Management Information and Reporting System

A comprehensive web-based school management system for elementary schools.

## Features

- Student Academic Management
- Attendance Tracking (Student & Staff)
- Behavior & Conduct Tracking
- Enrollment Management
- Billing & Collections
- Health Records
- Budget Management
- Comprehensive Reporting & Analytics

## Technologies Used

- ASP.NET Core MVC (.NET 8)
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Chart.js

## Setup Instructions

1. Clone the repository
2. Update the connection string in `appsettings.json`
3. Run database migrations: `Update-Database`
4. Run the project: `F5` in Visual Studio

## Default Login Credentials

- Admin: `admin1` / `admin123`
- Principal: `principal1` / `principal123`
- Teacher: `teacher_ana` / `teacher123`
